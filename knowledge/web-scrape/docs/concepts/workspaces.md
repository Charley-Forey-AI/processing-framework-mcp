# Workspaces

Processing Framework allows users to define their workflow using FME Workspaces. A Workspace is an API resource that establishes an FME Workspace as an Operation within the Processing Framework.

After a Workspace is created, it must be tied to an [Operation](../operations/) to be used within a Procedure.

See the [How To Guide](../../how-to-guides/workspaces/basics/) for more information on the creation process.
