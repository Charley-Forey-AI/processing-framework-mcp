# Getting Started

The Processing Framework API is available through Trimble Cloud Console. The following steps are a short set of prerequisites to using the API.

## Prerequisites

Use the following getting started guides to create an application, subscribe to an API, and generate an access token:

1. [Create an application in Trimble Cloud Console](/content/getting_started/register/).
2. [Subscribe](/content/getting_started/subscribe/) to the Processing Framework API.

   | Environment | API Product | Proxy Name |
   | --- | --- | --- |
   | Production | [Trimble Cloud Processing Framework](https://console.trimble.com/home/consumer/products/2c9180847883247e0178899762d90023) | pfapi-1 |
   | Staging | [Trimble Cloud Processing Framework](https://console.stage.api.trimblecloud.com/home/consumer/products/2c9180847883247e0178899762d90023) | pfapi-1 |
3. [Generate an access token](/content/getting_started/authentication/).
4. Begin calling the Processing Framework API using your access token in the header of your API calls.
