# Processing Framework

Processing Framework is a service that orchestrates advanced processing in the cloud. Using the Processing Framework,
you can string multiple Operations to process files. The service supports processes including file conversions, data
aggregation, and others.

You can use Processing Framework workflows to complete a series of steps through a single execution of a Procedure. You
can structure Procedures in various ways and compose them of any combination of Operations that can be executed
sequentially or in parallel based on the data processing requirements.

The Processing Framework works in tandem with your application and data storage. The first Operation involves retrieving
source material from a location you have defined. The final Operation involves posting the processed data to a location
for your application to access.
